# 项目管理

学习如何在 yflow 中创建和管理翻译项目。

## 创建项目

### 通过管理界面

1. 登录管理后台
2. 导航到「项目」页面
3. 点击「新建项目」按钮
4. 填写项目信息：
   - **名称**：项目显示名称
   - **描述**：项目用途说明
   - **默认语言**：主语言，其他语言与之对照翻译

### 项目结构

```
my-project/
├── src/
│   └── locales/
│       ├── en.json      # 默认语言
│       ├── zh-CN.json   # 简体中文
│       └── ja-JP.json   # 日语
└── .i18nrc.json         # CLI 配置文件
```

## 管理项目

### 编辑项目

在项目列表页面，点击项目可编辑：
- 修改项目名称和描述
- 更改默认语言（会影响所有翻译对照）

### 归档项目

不再需要的项目可以归档：
1. 进入项目详情
2. 点击「归档」按钮
3. 归档后项目仍可查看，但不参与活跃统计

### 删除项目

::: warning
删除项目会同时删除所有翻译数据，此操作不可恢复。
:::

## 配置语言

### 添加语言

1. 进入「语言管理」页面（需要管理员权限）
2. 点击「添加语言」
3. 选择或输入语言代码：
   - 常用：`en`, `zh-CN`, `zh-TW`, `ja-JP`, `ko-KR`
   - 格式：IETF 语言标签

### 设置项目语言

1. 进入项目详情
2. 在「语言」标签页添加/移除语言
3. 设定默认语言

### 语言配置示例

```json
// .i18nrc.json
{
  "projectId": 1,
  "languages": ["en", "zh-CN", "ja-JP"],
  "defaultLanguage": "en"
}
```

## 项目权限

yflow 支持三级项目权限：

| 权限 | 描述 |
|------|------|
| **所有者 (Owner)** | 完全控制，可删除项目、管理成员 |
| **编辑者 (Editor)** | 可以编辑翻译、导入导出 |
| **查看者 (Viewer)** | 只读权限 |
