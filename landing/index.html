CLI Workflow

从代码到云端，
只需三步

01

yflow init

初始化配置文件，设置项目 ID、API 地址、语言映射。

.i18nrc.json
02

yflow import

扫描 messagesDir 目录，自动解析 JSON 文件中的翻译 key，推送到服务器。

03

yflow sync

从服务器拉取最新翻译，保持本地文件与云端同步。

Terminal
$ yflow init
✓ 配置文件已生成: .i18nrc.json
$ yflow import
